# Atchutchi Ferreira

**ICT e empreendedorismo**

Construo produtos digitais e sistemas para empresas e instituições.

## O que faço

- 🔧 Liderança técnica e entrega
- 💻 Desenvolvimento web e integração de dados
- 📊 Plataformas com dashboards, relatórios e automação
- 📈 Projetos orientados a negócio

## Tecnologias principais

**Backend:**
- TypeScript, JavaScript, Python
- Django REST Framework, JWT, Celery
- Docker Compose, PostgreSQL, Redis, Nginx

**Frontend:**
- Next.js, Tailwind CSS
- Zustand, ECharts

**Ferramentas:**
- PDF e Excel export com WeasyPrint e openpyxl
